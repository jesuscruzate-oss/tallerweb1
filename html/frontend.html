Tecnologías de Frontend

El Frontend es la parte de una aplicacion web que los usuarios pueden ver e interactuar directamente

Tecnologias principales

  1. HTML

    Define la estructura del contenido de una pagina web.

  2. CSS

    Permite controlar la apariencia visual del sitio.

  3. JavaScript

    Permite crear interfaces interactivas.
